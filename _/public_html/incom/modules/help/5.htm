Описание:

Система Incom CMS 2.0. позволяет осуществлять быстрый доступ к управлению сайтом из административного раздела с помощью административной панели, содержащей набор кнопок для выполнения стандартных операций над элементами сайта:

Разделы системы – выпадающий список, который позволяет выбрать и перейти к редактированию того или иного раздела/подраздела:

Выбор языка – выпадающий список, который позволяет выбрать языковую версию сайта:

Помощь – раздел позволяет просмотреть инструкцию по управлению административной частью сайта:

Синхронизация поиска – при добавлении, удалении, редактировании информации на сайте необходимо обновление информации в базе данных. После обновления функция поиска будет выдавать наиболее актуальную информацию.

Чтобы синхронизировать обновленные данные в системе управления сайтом и базой данных нужно просто нажать кнопку «Поиск», расположенную в левом верхнем углу активного окна:
