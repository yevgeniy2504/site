1. Справочная документация 

1.1. Основные сведения

Раздел «Помощь» в административной части предназначен для пользователей продукта Incom CMS 2.0 (Система управления Контентом, Incom Content Management Systems).

Данный раздел представляет собой подробное описание интерфейса системы, ее основных модулей и наиболее типичных операций по управлению Интернет-проектом. Также раздел «Помощь» содержит полное описание прикладного программного интерфейса системы и может использоваться программистами при разработке сайтов и заказных модулей.

Для управления сайтом вам не потребуются дополнительные услуги специалистов в области веб-дизайна и веб-программирования. Каждому программному модулю посвящена отдельная статья, в которой дается подробное описание руководства со скриншотами.

С помощью подробного описания интерфейса системы управлять веб-сайтом сможет штатный сотрудник компании, обычный пользователь персонального компьютера без специальных навыков программирования и HTML-верстки.

1.2. Руководство по установке

Установка продукта состоит из 3-х шагов

1. Ознакомление с лицензионным соглашением

2. Настройка подключения к базе данных и инсталлирование баз данных.

Для подключения к базе данных необходимо заполнить поля:

хост (хост базы данных по умолчанию это localhost), имя базы данных, логин и пароль базы данных, имя, фамилия, логин и пароль администратора (для создания первой учётной записи).

3. Авторизация и переход в административную часть, просмотр публичной части сайта.
