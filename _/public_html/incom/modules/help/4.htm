4.1. Пользователи

Описание модуля

Система Incom CMS 2.0. позволяет создавать произвольное количество групп пользователей с разными правами доступа к ресурсам сайта. С помощью настройки прав доступа вы можете: управлять доступом пользователей к информации в публичном разделе сайта, управлять доступом пользователей к "закрытым" разделам сайта. Например, закрытому форуму, партнёрскому разделу или любому другому разделу, требующему ограничения доступа, разрешать пользователям изменять некоторые файлы и папки сайта, запретив при этом редактирование других файлов:

Экспорт данных в MS Excel

Функция предназначена для экспорта данных из информационного блока в файл MS Excel. При клике на «Export» выводится информация о результатах выгрузки данных, а также отображается ссылка для выгрузки файл MS Excel на локальный компьютер:

Список пользователей

При клике на раздел «Список пользователей» отображается таблица всех существующих пользователей. Есть возможность добавления, удаления, редактирования, активизации, деактивации (блокировки) пользователей системы. Также имеется фильтровки всего списка по определённым значениям (кнопка «дополнительно») и экспорта данных в файл MS Excel.

Ссылка «Настроить» служит для настройки полей пользователей, даёт возможность добавлять и редактировать поля.

Для того, чтобы добавить нового пользователя, необходимо кликнуть на ссылку «Добавить пользователя», в открывшемся окне появятся поля для заполнения, таких, как активен/неактивен, пользовательская группа и т.д.

Для редактирования пользователей необходимо выбрать в контекстном меню раздел «редактировать» или сделать двойной щелчок по строке того пользователя которого нужно отредактировать.

В данном разделе имеется возможность поменять логин и пароль для входа в административную часть, добавить e-mail, поменять пользовательскую группу, уведомить пользователя о регистрации (отправка письма на e-mail).

Настройка полей

Настройка полей позволяет перейти к настройке внешнего вида полей формы:

В открывшемся окне появится форма настройки полей:

• обязательное поле для заполнения – выбираем: будет ли данный пункт web-формы обязательным для заполнения,

• выделить поле – функция визуального выделения поля web-формы;

• ID сортировка - порядковый номер для последовательного отображения пункта меню;

• наименование поля - описание вносимой информации в соответствующее поле, например, «Ф.И.О.»;

• наименование поля eng - вносится только на латинице, например, «name»;

• тип поля - позволяет сгенерировать поле вывода:

Тип поля «Текстовое (text)» - предназначен для ввода текстовой информации.

Тип поля «Текстовое (дата)» - предназначен для ввода даты.

Тип поля «Текстовое (логин)» - предназначен для ввода логина (имени пользователя).

Тип поля «Текстовое (пароль)» - предназначен для ввода пароля.

Тип поля «Текстовое цена (числовое)» - предназначен для вывода стоимости.

Тип поля «Много текста (textarea)» - предназначен для ввода блока текстовой информации.

Тип поля «Выбор (checkbox)» - предназначен для выбора из нескольких заранее определенных позиций.

Тип поля «Список (select)» - предназначен для выбора одного значения из фиксированного выпадающего списка значений.

Тип поля «Список (radio button)» - предназначен для выбора одного из значений. При этом все остальные поля данного типа автоматически станут невыбранными.

Тип поля «html редактор» - предназначен для вывода Визуального редактора.

Тип поля «Рисунок (file)» - предназначен для размещения рисунков.

Тип поля «Файл (file)» - предназначен для размещения прикрепленных файлов.

В зависимости от выбора типа поля добавляется еще некоторое количество полей, таких, как длина поля, формат даты, длина, высота, элементы выпадающего списка, размер, формат.

Группа пользователей

Данный раздел позволяет создавать пользователей и группы пользователей (с назначением привилегий и прав).

При клике на раздел «Группа пользователей» отображается таблица, где расположены все группы пользователей. Есть возможность добавления, удаления, редактирования, активизации, деактивации (блокировки) группы пользователей системы, также можно отфильтровать весь список по определённым значениям (кнопка «дополнительно») и сделать экспорт в MS Excel.

Для того, чтобы добавить новую группу, необходимо кликнуть на ссылку «Добавить группу» в открывшемся окне появятся поля для заполнения такие, как активна/неактивна, наименование группы, привилегии на разделы (доступ/ограничение к управлению данными разделами системы).

Для редактирования группы пользователей необходимо выбрать в контекстном меню раздел «редактировать» или сделать двойной щелчок по строке того пользователя, информацию которого нужно отредактировать.

4.2. Настройки продукта

Описание модуля:

Раздел Настройки продукта служит для управления настройками сайта в целом. Для удобства работы раздел разделен на подразделы: шаблоны сайта, языковые версии, обработка адресов, обработка логов. Каждая закладка содержит логически сгруппированный набор полей, позволяющих выполнять определённые действия по настройке выбранного модуля.

Шаблоны сайтов

Управление шаблонами, на основе которых выполняется отображение страниц сайта, осуществляется в форме Шаблоны сайта. С помощью этой формы вы можете создавать новые, изменять и удалять существующие шаблоны сайта. Работа с шаблонами требует знания HTML.

При клике на раздел «Шаблоны сайтов» отображает таблицу, где расположены все шаблоны сайтов публичной части. Есть возможность просмотра, добавления, удаления, редактирования, активации, деактивации (блокировки) шаблона.

При изменении дизайна сайта или добавлении новой языковой версии создается новый шаблон.

Для редактирования шаблона необходимо дважды кликнуть на строку или воспользоваться контекстным меню.

В открывшемся окне появляются поля для заполнения:

• активен – статус отображения шаблона сайта;

• языковая версия шаблона – весь список доступных языковых версий;

• наименование шаблона – произвольное название шаблона;

• наименование папки – наименование директории на сервере, в которой будет храниться весь шаблон (заполняется только на латинице, без спецсимволов);

• заголовок, описание, ключевые слова, необходимые для лучшей цитируемости в поисковых системах (мета-теги, которые будут отображаться по умолчанию на каждой странице);

• внешний вид шаблона – позволяет перейти к настройке внешнего вида шаблона сайта (Функция позволяет вставлять весь исходный код дизайна шаблона, исключая рабочую область, вместо неё должна находиться строка «#WORK_AREA#»);

• файл стилей – данное поле служит для описания таблиц каскадных стилей (CSS), используемых на страницах сайта.

Для редактирования шаблона необходимо выбрать в контекстном меню раздел «редактировать» или сделать двойной щелчок по строке того шаблона, который нужно отредактировать.

Тег «title» — заголовок страницы

Этот тег очень важен для индексации поисковыми машинами. Он является одним из основных источников ключевых слов для поисковика. Он показывается вверху окна браузера и не очень важен для посетителя, хотя является единственным «названием» или «заголовком» страницы сайта. Он же показывается в списке ссылок — результате поиска в поисковых машинах.

Важно:

Длина заголовка не должна быть большой — не более 80-100 символов.

Самые важные ключевые слова должны стоять ближе к началу.

Не используйте спецсимволы, они не нужны ни посетителю, ни поисковику.

Не используйте только заглавные буквы, поисковые машины это не любят.

 

Изменить этот тег можно в поле «Заголовок» подраздела «Шаблоны сайтов» модуля настройки продукта.

Тег «description» — описание

Этот тег также является источником ключевых слов для поисковых систем. Текст отсюда показывается в результатах поиска и является определяющим для посетителя — заходить на страницу или нет.

Важно:

Длина должна быть не более 200 символов.

Текст должен быть понятен посетителю и содержать важные ключевые слова и фразы.

Не используйте спецсимволы и заглавные буквы.

 

Изменить этот тег можно в поле “Описание“ подраздела «Шаблоны сайтов» модуля настройки продукта.

Тег «keywords» — ключевые слова

Этот тег полезен только для некоторых, обычно старых, поисковиков и каталогов. Но даже с них есть небольшой поток посетителей, поэтому имеет смысл их заполнить.

Важно:

Длина должна быть не более 200 символов.

Ключевые слова и фразы нужно разделять запятыми.

Не используйте спецсимволы и заглавные буквы.

Изменить этот тег можно в поле «Ключевые слова» подраздела «Шаблоны сайтов» модуля настройки продукта.

Языковые версии

В разделе «Языковые версии» отображается таблица, где расположены все языковые версии:

Существует возможность добавления, удаления, редактирования, активации, деактивации (блокировки) выбранного языка.

Для того, чтобы добавить новый язык, необходимо кликнуть на ссылку «Добавить язык», в открывшемся окне появятся поля для заполнения:

активен - признак активности. Если язык не активен, то отображение сообщений на данном языке не доступно;

язык по умолчанию - если в коде страницы каким-либо образом не определён язык или не один язык не активен, то программа будет выбирать язык по умолчанию;

наименование – произвольное название языка;

сокращение translate - аббревиатура языка на латинице (Желательно вносить не более 3-х символов).

Обработка адресов

В разделе «Обработка адресов» отображается таблица, в которой находятся все заблокированные ip-адреса, при ошибочном вводе авторизующих данных (логина и пароля) более 3-х раз. В данной таблице хранится дата, время подбора паролей, ip-адрес, браузер и операционная система.

С помощью этой формы вы можете удалять существующие в списке IP-адреса.

Пользователи, чьи IP-адреса попадают в данный «черный» список, ограничены в доступе к административной части системы.
