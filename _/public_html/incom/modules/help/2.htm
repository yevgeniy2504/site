2. Общая информация

2.1. Начало работы и авторизация

Управление сайтом осуществляется через административный раздел, доступ к которому есть только у зарегистрированных пользователей. Для обычных посетителей сайта доступ в административный раздел закрыт.

Для работы в административном разделе необходимо авторизоваться. Для этого следует выполнить следующие действия:

Откройте в браузере страницу http://адрес вашего сайта/incom/. Все административные файлы сайта находятся в каталоге /incom/.

В форме авторизации введите ваш Логин (имя доступа) и Пароль (Данные авторизации могут быть изменены в разделе «Пользователи»). Есть возможность запоминания учетной записи на текущем компьютере, для того, чтобы не проходить авторизацию повторно (данная возможность активна в течение 3-х дней).

После нажатия кнопки «Далее» осуществляется проверка правильности введенных данных, при наличии таких Логина и Пароля в базе авторизационных данных пользователь автоматически перенаправляется в административную часть сайта. При неверном вводе авторизационных данных выводится ошибка «Неправильно введён логин или пароль». Третий неверный ввод данных ведет за собой блокирование ip-адреса компьютера пользователя, а в cookies (временные файлы Internet) добавляется соответствующая запись, которая блокирует возможность авторизации на сайте.

2.2. Завершение работы

Для того, чтобы закончить сеанс и перейти в режим неавторизованной работы, нажмите кнопку Выход, расположенную на панели управления в административном разделе.
